# SMSApi

Handler for sending SMS through SMSApi.

## Configuration

Options for _from_ are describen in documentation of SMSApi - it could be f.e. ECO or name defined in panel.

```json
{
  "smsapi": {
    "username": "<username>",
    "password": "<password for api>",
    "from": "<type of message>",
    "to": "<number>",
    "test": "false"
  }
}
```
